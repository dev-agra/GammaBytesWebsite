PharmaFriend

Website for comparing pricing of medicines on multiple online platforms. Tech Stack: Python, Django, Selenium

Welcome to PharmaFriend, your one-stop destination for all your medical needs! We are committed to providing you with the best prices for medicines by comparing prices from different websites, so you can make an informed decision about your purchase. But that's not all - we offer consultation services with licensed doctors and pharmacists to provide you with the right guidance for your medical needs. PulmoDetect

Comprehensive system for quick diagnosis and aid in Tuberculosis Detection Tech Stack: Python, TensorFlow, Django, JavaScript

In order to bridge the gap between Deep Learning researchers and medical professionals we develop a very accessible free prototype system which can be used by medical professionals to understand the reality of Deep Learning tools for chest X-ray diagnostics. The system is designed to be a second opinion where a user can process an image to confirm or aid in their diagnosis. Code and network weights are delivered via a URL to a web browser (including cell phones) but the patient data remains on the users machine and all processing occurs locally.

The system is built upon a deep convolutional neural network (CNN) architecture, trained on a diverse dataset of chest X-ray images from patients with confirmed TB cases and healthy individuals. PulmoDetectTB employs state-of-the-art image processing techniques to extract meaningful features from the X-ray images, allowing for the precise identification of TB-related abnormalities.
